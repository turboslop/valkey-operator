# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |

## Reporting a Vulnerability

Please report security vulnerabilities by opening an issue at
https://github.com/turboslop/valkey-operator/issues
with the label `security`.

Do NOT open public issues for critical vulnerabilities.
Instead, email the maintainers directly.
